# Supabase Configuration

Этот каталог содержит SQL скрипты и документацию для настройки базы данных Supabase.

## Файлы

### `init.sql`
Полный SQL скрипт для инициализации базы данных. Включает:
- ✅ Создание 4 таблиц (users, workouts, exercises, workout_sets)
- ✅ Индексы для оптимизации запросов
- ✅ Row Level Security (RLS) политики
- ✅ Триггеры для автоматического обновления timestamps
- ✅ Function для автоматического создания профиля пользователя
- ✅ Views для удобного получения данных
- ✅ Комментарии и документация

### `SETUP.md`
Пошаговая инструкция по настройке Supabase проекта:
- Как запустить SQL скрипт
- Проверка создания таблиц и политик
- Настройка Authentication
- Получение API ключей
- Тестирование базы данных
- Troubleshooting

## Быстрый старт

1. Откройте [Supabase Dashboard](https://app.supabase.com)
2. Выберите ваш проект
3. Перейдите в **SQL Editor**
4. Скопируйте и запустите содержимое `init.sql`
5. Следуйте инструкциям в `SETUP.md` для полной настройки

## Структура базы данных

```
users → workouts → exercises → workout_sets
```

- **users**: Профили пользователей (extends auth.users)
- **workouts**: Тренировочные сессии
- **exercises**: Упражнения внутри тренировки
- **workout_sets**: Подходы для каждого упражнения

Все таблицы защищены Row Level Security - пользователи видят только свои данные.

## Следующие шаги

После настройки базы данных:
1. Получите API ключи из Supabase Dashboard
2. Установите Supabase Swift SDK в iOS проект
3. Создайте `SupabaseService.swift` для интеграции
4. Реализуйте авторизацию и CRUD операции
